# Cities

simple application which shows cities, update and create new city

## Used technologies:

- React (hooks)
- Typescript
- Material-Ui
- axios
- json-server

## Start project

Clone repository

`git clone https://github.com/x4AEKx/cities.git your-folder`

For run project:

`yarn dev`
